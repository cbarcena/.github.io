What is Frontend Development?

Front End Developers have to understand the relationship of the entire site, how to link pages and how they relate. They must possess HTML mastery and consider structural design of a website at all times. Frontend Developers work primarily with HTML, CSS, and JavaScript. They build and maintain content for the web, targeting different platforms such as desktops, tablets, handhelds, etc. They are responsible for design and functionality, layout, typography, images and other visual elements and styling. They must know how to create HTML Prototypes, have experience with JQuery, gitHub, Grunt.js, CSS preprocessors, JavaScript, HTML, Node.Js, Indexeddb, Git, Templating Languages, and Ajax.

Frontend Deverlopers can often refer to usability and user experience design as well. Additional Know-how: CSS, UX Design, Image editing, sketching, JavaScript, gitHub, grunt.js, css preprocessors and HTML mastery is a definite must.
