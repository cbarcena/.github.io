The role of HTML !!!CLEAN UP!!!

Hyper Text Markup Language or better known as HTML, can be described as the tags that define the rule that each group of content will have. MarkUp Language, meaning it “Marks up” content and how it relates to elements on the page.

HTML is responsible for structuring documents and identifying elements within a webpage. HTML Makes a page readable and conveys structure to the user. It is identified by wrapping within a <html>tag

DOCTYPE

The <!DOCTYPE HTML> located at the top of each HTML document tells the browser or user agent which version of HTML you are using and which version of syntax to use in parsing.

Opening and Closing Tags

HTML is the standard language used for structuring the web. It’s the number one source for apprehension for web designers especially for visual and print designers. Every webpage consists of three main tags. Each with opening and closing tags:

  1. <html></html>
  2. <head></head>
  3. <body></body>

A tag can contain a name, attribute and value. The equal sign indicates an assignment operator, then enclose the value within quotation marks.
