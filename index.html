What is web development?

Web development involves the understanding of the web, how it works, how to create HTML, creating basic CSS, understanding SEO (search engine optimization) and building websites. There are three core languages that every developer must know; HTML, CSS and JavaScript. HTML is the most important as it is the essential structure of all websites, CSS is the page design and JS is the page behavior.

There are several levels of web development, such as, frontend development, backend development, and full stack developers, each with a specific skill set. At each level, understanding the basics and mastering HTML is imperative in being successful within the industry.
