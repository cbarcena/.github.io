Let's Get Started

As a developer you must understand the differences between the internet and the web. The internet is a massive network that shares and transmits information, such as email, sms, app data, web, etc. The “web” is a series of interactions between two types of systems. First being Clients which makes requests and renders the content, for example, browsers, apps, screen readers, content aggregators, etc. Second, servers which are applications that deliver web content and services to clients.

Frontend Developers work primarily with HTML, CSS, and JavaScript. They build and maintain content for the web, targeting different platforms such as desktops, tablets, handhelds, etc. A Backend Developer works with languages that render in the client browser. Working with the files that render in the background, where most of the action is not happening in front of the user. As for a Fullstack developer, these developers are considered the “unicorns” because they work with all different layers of development. We’ll go more in depth on both Frontend Developers and Backend Developers on the next couple pages.
