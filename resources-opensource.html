Open Source Websites

Modern tools are powerful however becoming more complex, and the use of web components have helped developers with open source content that can be easily manipulated to create what they need. Polymer v1.0 is a lightweight layer of web components, making it easier to build a reusable web component or custom element. Polymer offers an App Toolbox which is a progressive web app that includes app level components, web components, layout routing and internationalization and a command line tool to build and deploy applications. As a user you can expect Polymer and PRPL Pattern to increase the page load 2x its normal load time and 6x less data than that of a native app.This helps developers with deploying rapid prototyping all the way though.

Some additional open source resources that will be of good use are GitHub, a respository website and CodePen, where developers can see exemplified work from other developers.
