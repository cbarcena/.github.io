Bootstrap

Frameworks and Boilerplates:

  • Pre-built collections of CSS, HTML and JavaScript
    files designed to speed up development of sites.

BoilerPlate:

  • A set of templates built around a specific starting point or goal
  • Meant to provide a starting point for building sites and or apps

Framework:

  • A collection of assets designed to help build sites or applications
  • This could include CSS guides, JavaScript libraries or helpful scripts and html templates
  • Frameworks are often used to build HTML prototypes.

Complex UI Frameworks

  • Built around developing the front end of sites or applications
  • UI framework function move like development tools
  • Give developers widest set of tools to work with

Minimal Frameworks

  • Focus on a stable base of a project to start from, leave the back of the styling up to the designer
  • Designed to have small footprint and help out in specific ways
  • The type of framework right for you will be based on personal philosophy and project type

Prototyping

  • Frameworks are often used to build HTML prototypes.
  • Providing working prototypes in hours rather than days
  • Helps iteration framework code is not used for final production
