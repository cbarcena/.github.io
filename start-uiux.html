UI and UX Designers

Understanding the differences between a user interface and user experience designer is essential when seeking employment within the tech industry. For starters, a web designer is a designer focused on making a website visually pleasing with little to no user interaction. Where as a user interaction (UI) designer focuses more on how a product is laid out and a user experience (UX) designer is concerned with how a product feels. Both of which focus more on the functionality of a website, where and what is to happen when a user makes a selection, hits a button or clicks a link. These can be anything such as web forms, contact us pages, user login or anything that requires the user to interact with the website.

Learning the basics of web design is the foundation to successful implementation of skills for both a UI and a UX designer. To learn more about the differences between UI and UX take a look at www.fastcodesign.com
