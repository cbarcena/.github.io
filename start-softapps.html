Software & Web Applications

It's not always necessary to build a website with all the nics and crannies of corporate level development. When a website requires the interaction from a user you can instead build a web application focusing more on the interactivity of the user as opposed to a wordy website loaded text content. One example of a web app I found to be very creative, is SF Coffee Finder. The creativity in both the design and interactivity makes the user want the app and encourages others to join.

As for an information only website, West Fresno Family Resource Center provides all the needed text content and imagery for the community and those looking to learn more about the company.

In any case, finding the right software to help you create and build a robust website that meets your design and functionality needs can be done so by downloading free text editors, and by taking advantage of free web applications designed to fulfill those minor details such as imagery and graphic edits. Below are just a few of the more popular text editors and web applications available.
