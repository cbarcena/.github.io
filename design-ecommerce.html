Using WordPress for Ecommerce See also Marketing

Ecommerce is incredibly complex, its important to choose a developer over features. When considering a build that involves ecommerce, one of the most popular platforms to choose from is WordPress. Take a look at some helpful advice, resources and what you should consider when developing an ecommerce website below:

  • Functionality
    • Order management
    • Multiple payment gateways
    • Live shipping estimates
    • Digital file delivery etc.
  • Different Types of Platforms
    • Open source: GPL licensed, can be viewed/modified freely
    • Freeform platforms: core functionality free, advanced niche functionality paid
  • Platform extensions and providers:
    • BuiltWith.com: Ecommerce usage stats
    • WooCommerce
    • EasyDigitalDownload.com
    • WPEcommerce.com

Implementing third-party platforms

  • Hosted platforms
    • You don’t know or manage the code
    • Easier setup
    • Harder to customize
  • Providers
    • GumRoad.com: primarily for digital files and videos
    • ClickBank.com: affiliate payments
    • Shopify.com

Securing your site

  • Understanding the basics of secure socket layer (SSL)
    • Without it: Site owner is held financially liable, risking loss of customer trust
      • A site that looks secure increases conversions
      • All sensitive info should be entered on a secure page (passwords, cc’s etc.)
    • Https: encrypts web data
    • SSL: secure socket layer
    • Installing SSL many hosts do this for a small fee
    • PCI DSS Compliance: Payment Card Industry and Data Security Standard for handling credit card numbers More Info: https://goo.gl/gzhkrU
    • Positive Factors: ecommerce platform, hosting environment, organizational practices

Web Hosts/Providers

  • WooCommerce.com
  • WPEngine.com

Protecting sensitive data

  • Configure SSL
    • Use .htaccess
    • Manage through platform
    • Install plugin
  • Https://sslredirection on wordpress
  • Protect admin login page
    • JetPack.com: simple sign on feature
    • GetClef.com: 2 factor authentication

One must always consider an Outline for an added security plan customized to your site
